# Lilypad-gloves-adxl

![GitHub Logo](/lillypad-arduino.jpg)

Control your smartcar from hand movement 
# Component 
 * Arduino Uno R3
 * HT12E and HT12D IC
 * Lilypad/Nano
 * 5 V Battery/ Breadboard Power Supply
 * Jumper wires
 * 9v Battery
 * L293D IC
 * Plain PCB
 * Breadboard
 * ADXL335 Sensor
 
 
 It's very easy to make get all components. You can visit [IOTWEBPLANET!](https://iotwebplanet.com)
