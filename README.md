# Lilypad-gloves-adxl
Control your smartcar from hand movement 
# Component 
 * Arduino Uno R3
 * HT12E and HT12D IC
 * Lilypad
 * 3.7 V Battery
 * Jumper wires
 * 9v Battery
 * L293D IC
 * Plain PCB
 * Braedboard
